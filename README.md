Proposta de Valor
O ReCiclo tem como objetivo conectar pessoas que desejam doar objetos com outras que precisam deles, promovendo sustentabilidade, redução de desperdício e impacto ambiental positivo.

1. Tela Home / Feed
Apresenta uma lista de itens disponíveis próximos ao usuário, com imagem, nome e distância.
2. Tela de Filtros
Permite filtrar os itens por categoria, distância e estado de conservação.
3. Tela de Detalhes
Exibe informações completas do item, incluindo descrição, imagem e dados do doador.
4. Tela de Cadastro de Item
Permite ao usuário cadastrar um novo item para doação.
5. Tela de Chat
Permite a comunicação entre doador e interessado para combinar retirada.
6. Tela de Perfil
Mostra informações do usuário e suas conquistas (Selos Eco).

🔄 Jornada Principal
1.	Usuário abre o aplicativo
2.	Visualiza itens disponíveis no feed
3.	Aplica filtros (opcional)
4.	Seleciona um item
5.	Visualiza detalhes
6.	Inicia conversa com o doador
7.	Combina retirada do item
8.	Conclui a ação e recebe um selo Eco

Impacto
O sistema incentiva a reutilização de materiais, reduzindo o descarte de lixo e promovendo a sustentabilidade.
